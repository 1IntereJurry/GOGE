About the Project

GOGE is a handcrafted C# console application that bridges the gap between classic terminal aesthetics and modern learning mechanics. It was developed to showcase the power of the .NET environment within a simple yet engaging text-based interface.

Key Features:

  • Classic CLI atmosphere
  • Interactive gameplay mechanics
  • Lightweight and portable
  • Pure C# source code
